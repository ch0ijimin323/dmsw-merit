# 화면디자인 수행평가

HTML / CSS로 제작한 화면디자인 수행평가 과제물입니다.

## 🔗 사이트 바로 실행하기

### 👉 https://ch0ijimin323.github.io/dmsw-merit/

위 링크를 클릭하면 로그인 화면부터 사이트가 바로 실행됩니다.

## 폴더 구성

```
├── pages/   HTML 페이지
│   ├── p1.html   로그인
│   ├── p2.html   메인페이지
│   ├── p3.html   상점
│   ├── p4.html   벌점
│   └── p5.html   내 정보
└── css/
    └── style.css   공통 스타일시트
```

## 실행 방법

`pages/p1.html` 파일을 웹 브라우저로 열면 됩니다. 페이지 내 링크를 따라 각 화면으로 이동할 수 있습니다.
